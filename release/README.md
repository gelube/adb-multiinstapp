## 使用方法

### 快速开始
1. 双击 `启动.bat` 运行程序
2. 在"设备发现"标签页扫描设备
3. 在"应用安装"标签页选择 APK 和输入包名
4. 点击"开始安装"

### 使用说明
1. **设备发现**: 输入 IP 范围（支持跨网段），点击"开始扫描"
2. **应用安装**: 选择 APK 文件，输入包名，选择设备，点击"开始安装"
3. **失败重试**: 安装失败的设备会自动出现在此页面，可单独重试
4. **应用卸载**: 输入包名，选择设备，点击"开始卸载"

## 系统要求

## 常见问题

### Q: 扫描不到设备？
A: 确保设备已开启 USB 调试和无线调试，且与电脑在同一局域网。

### Q: 安装失败？
A: 查看"失败重试"标签页的失败原因，可单独重试失败的设备。

### Q: 版本显示不正确？
A: 点击"刷新版本"按钮手动刷新，或确保输入正确的包名。
